Cron Setup

CryptoLens uses system cron for reliable scheduling. Add these entries to your crontab:

Data Collection

# Fetch latest candles every minute (required)
* * * * * cd /path/to/cryptolens && venv/bin/python scripts/fetch.py >> logs/cron.log 2>&1

# Fill data gaps every hour (recommended)
0 * * * * cd /path/to/cryptolens && venv/bin/python scripts/fetch.py --gaps >> logs/cron.log 2>&1

Analysis

# Scan for patterns every 5 minutes (optional - or use Quick Actions)
*/5 * * * * cd /path/to/cryptolens && venv/bin/python scripts/scan_patterns.py >> logs/cron.log 2>&1

# Refresh statistics every 15 minutes (optional - or use Quick Actions)
*/15 * * * * cd /path/to/cryptolens && venv/bin/python scripts/compute_stats.py >> logs/cron.log 2>&1

Maintenance

# Clean old logs daily at midnight
0 0 * * * cd /path/to/cryptolens && venv/bin/python scripts/cleanup.py >> logs/cron.log 2>&1

# Verify data integrity daily at 3 AM
0 3 * * * cd /path/to/cryptolens && venv/bin/python scripts/db_health.py --fix >> logs/cron.log 2>&1

Quick Reference

Cron Format:

minute hour day month weekday command
  • * = every unit
  • */5 = every 5 units
  • 0 = at unit 0

Common Schedules:

  • * * * * * = Every minute
  • */5 * * * * = Every 5 minutes
  • 0 * * * * = Every hour
  • 0 0 * * * = Daily at midnight

Edit your crontab with: crontab -e  |  View logs: tail -f logs/cron.log
