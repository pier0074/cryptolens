Cron Setup

CryptoLens uses system cron for reliable scheduling. Add these entries to your crontab: 

# CryptoLens - Fetch candles every minute
* * * * * cd {{ request.host_url.rstrip('/').replace('http://', '/path/to/').replace('https://', '/path/to/') }} && venv/bin/python scripts/fetch.py >> logs/cron.log 2>&1

# CryptoLens - Fill data gaps every hour
0 * * * * cd {{ request.host_url.rstrip('/').replace('http://', '/path/to/').replace('https://', '/path/to/') }} && venv/bin/python scripts/fetch.py --gaps >> logs/cron.log 2>&1

Edit your crontab with: crontab -e
